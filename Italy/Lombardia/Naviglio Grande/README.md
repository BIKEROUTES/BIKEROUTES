# Naviglio Grande

Ride along the [Naviglio Grande](https://en.wikipedia.org/wiki/Naviglio_Grande)
from Abbiategrasso to Tornavento, then along the Ticino up to Sesto Calende, at
the Southern tip of Lake Maggiore.
